Created a sample log file with INFO, WARNING, and ERROR entries.

Created a shell script to count log entries, count INFO/WARNING/ERROR messages, find the latest ERROR, and generate a report.

Made the script executable.

Executed the script to display log summary and generate report.

./log_analyzer.sh system.log > output.txt:
